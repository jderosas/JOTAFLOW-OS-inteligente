# Modulo SCAN12

SCAN12 es el diagnostico breve de entrada de JOTAFLOW OS.

Su funcion es detectar el primer punto inteligente de mejora de una empresa, equipo o profesional, usando 12 preguntas ordenadas en tres dimensiones:

- Direccion
- Operacion
- Crecimiento

El formulario vigente esta en:

```text
Formulario-SCAN12-v1.0.md
```

Este modulo agrega la estructura operativa necesaria para convertir el formulario en un sistema aplicable:

| Archivo | Funcion |
|---|---|
| `spec.md` | Define que busca lograr SCAN12, para quien existe y que resultado debe producir. |
| `protocol.md` | Ordena el proceso repetible desde la respuesta del formulario hasta la devolucion. |
| `tasks.md` | Divide el modulo en tareas concretas y verificables. |
| `scoring.md` | Define como puntuar respuestas y leer resultados. |
| `interpretation-matrix.md` | Traduce puntajes en lecturas y posibles UMOs. |
| `output-report-template.md` | Plantilla base para entregar la devolucion ejecutiva. |
| `agents.md` | Define agentes posibles para asistir el proceso sin reemplazar validacion humana. |
| `evidence.md` | Registra aplicaciones reales y resultados observados. |
| `learning-log.md` | Registra aprendizajes para nuevas versiones. |
